参加申込フロントエンド

参加申し込みフロントエンドは、通常のPerlによるCGIとして動作するため、 一般的なhttpdサーバ環境を構築し、適切なディレクトリに参加申込フロントエンドGitリポジトリをcloneする。

調査中

企画申込フロントエンド

企画申し込みフロントエンドは、通常のPerlによるCGIとして動作するため、 一般的なhttpdサーバ環境を構築し、適切なディレクトリに企画申込フロントエンドGitリポジトリをcloneする。
Git clone後の手順は、リポジトリのREADME.mdを参照。
なお、以下のPerlバージョン/モジュールに依存するため、事前にインストールしておくこと。
Perl5.14以上

conkan企画管理サブシステム

conkan企画管理サブシステムは、Dockerコンテナとして動作するが、開発環境ではシステムそのものはDockerコンテナイメージの中には含まず、コンテナ起動時に-vでマウントする。
そのため、まずDockerコンテナ動作環境を構築し、任意のディレクトリにconkan企画管理サブシステムGitリポジトリをcloneする。
Git clone後の手順は、リポジトリのREADME.md、INSTALL.md、DocMemo下の資料をを参照。
